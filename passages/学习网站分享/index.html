学习网站分享

最近正在学习Web前端，在这里推荐几个很好的学习网站。

FreeCodeCamp

Free Code Camp 已经被证明是获取编程工作的最有效路径，事实上，没人完成了所有课程，因为他们都在完成之前就找到了工作。

网址freecodecamp.one ，这个网站的详尽的讲解了Web知识，从前端到后端循序渐进。

网站的学习特色是上手实操，学习时需要将知识点实际操作一遍才可以通关，学习的进程中也有实际的项目，让你为开源项目贡献代码，总共分为7个部分，通过详尽的课程实现前后端的精通。

MDN

全称为Mozilla Developer Network，是一个汇集众多Mozilla基金会产品和网络技术开发文档的免费网站。

源于开发者，服务开发者。

许多网络方面的文档都可以在这里找到，知识更加严谨。

网址 https://developer.mozilla.org/zh-CN/

Tencent Cloud Studio

相比刚开始使用时需要绑定一个主机的使用逻辑，现在这种完全免费的方式更适合正在学习的我们（白嫖党福利）

最极客的云端开发体验

用户可以通过 Cloud Studio 创建项目的工作空间, 它提供了完整的 Linux 环境, 并且支持自定义域名指向, 动态计算资源调整, 可以完成各种应用的开发编译与部署。Cloud Studio 还提供了分享开发环境的功能, 用户可以保存当前的开发环境,分享给团队的其他成员。

每个空间都有一个独立的Linux环境，同时，内置编辑器有预览模式，可以很好的观察自己的网页样式。

网址https://studio.dev.tencent.com/

Thimble

Thimble是Mozilla开发的一款在线HTML开发工具，你可以使用这个工具实时看到你的代码带来的改变，并且可以便捷的编辑CSS，Javascript，不仅如此，你还可以在这个平台进行项目的分享，让他人看到你的网页内容。

网址：https://thimble.mozilla.org/zh-CN/
